# VidClaw 官网设计方案

> 待 Boss 确认后实施。部署目标：Cloudflare Pages。

---

## 一、产品定位

**VidClaw** — 用 AI 批量生成视频的 Mac 桌面工具

- 基于 Seedance 2.0（即梦旗舰视频模型）
- 自然语言描述 → AI 改写 → 自动出片 → 一键下载
- 支持批量任务、多账号、积分管理

**目标用户**：短视频创作者、内容运营、MCN 工作室

---

## 二、官网结构（单页滚动）

### ① Hero 区
**标题**：用 AI 批量生产视频
**副标题**：基于 Seedance 2.0，自然语言描述即可出片。批量任务、多账号管理，Mac 原生桌面体验。
**CTA**：`下载 macOS 版` （主按钮）+ `查看功能`（次要按钮）
**视觉**：产品截图（深色主题的桌面应用界面），轻微浮动动画

### ② 核心功能区（3-4 个模块，左文右图 / 左图右文交替）

**模块 1：AI 提示词改写**
- 标题：`告诉 AI 你想要什么`
- 文案：你用中文描述场景，VidClaw 自动优化为 Seedance 2.0 最佳提示词结构。不用研究参数，说人话就行。
- 视觉：对话框截图，左边用户输入"让这个女生跳一段街舞"，右边 AI 输出结构化 prompt

**模块 2：Seedance 2.0 全模式**
- 标题：`文生视频 · 图生视频 · 多模态参考`
- 文案：覆盖 Seedance 2.0 全部生成模式。文字描述、单张图片、多图参考、视频动作迁移——一个工具全搞定。
- 视觉：四种模式的示意图/截图

**模块 3：批量任务**
- 标题：`一次想好，批量出片`
- 文案：描述你的系列内容，AI 自动拆分为多个任务并排队执行。你去喝杯咖啡，回来视频全好了。
- 视觉：批量任务面板截图，显示多个任务的状态进度

**模块 4：账号 & 积分管理**
- 标题：`积分一目了然`
- 文案：实时查看积分余额（赠送/购买/会员）。支持多账号切换，工作室多号运营无缝衔接。
- 视觉：积分卡片截图，四列数据清晰展示

### ③ 工作流演示区
**3 步流程图**（横向排列，带编号和箭头连接）：
1. `扫码登录` — 用即梦账号直接登录，无需额外注册
2. `描述需求` — 自然语言输入，AI 自动改写优化
3. `下载成片` — 自动生成完成，一键下载到本地

### ④ 技术栈区（简洁）
- `Seedance 2.0` — 字节跳动旗舰视频生成引擎
- `Electron + Playwright` — 原生桌面体验 + 浏览器自动化
- `DeepSeek` — 高性价比 AI 模型，用于提示词改写

## 四、设计方向

### 风格：Product Dark
- 深色背景（#0A0A0B 附近），但不用纯黑
- 品牌色从 VidClaw 桌面应用继承（靛蓝/紫色系）
- 截图区域用轻微发光/阴影增强产品感
- 文字用浅灰，不用纯白

### 排版
- 标题：大号，semibold，行高紧凑
- 副标题：中号，regular，颜色降一级
- 正文：14-16px，max-width 60ch 保证可读性
- 不用系统字体，选一个有辨识度的显示字体（如 Space Grotesk / Clash Display）

### 交互
- 滚动触发的 fade-in 动画（IntersectionObserver）
- 截图区域 hover 时轻微上浮
- 下载按钮有明显的 hover 反馈
- 不做过度动画，保持克制

### 禁忌（impeccable 规范）
- ❌ 不用 cyan-on-dark 渐变
- ❌ 不用 glassmorphism
- ❌ 不用 glow 边框
- ❌ 不用 bounce 动画
- ❌ 不在每个标题上放 icon

## 五、技术方案

| 项目 | 选择 |
|------|------|
| 框架 | 纯静态 HTML + CSS + 少量 JS（无构建工具） |
| 部署 | Cloudflare Pages |
| 下载链接 | ClawHub release URL（待确认格式） |
| 域名 | 待定（vidclaw.ai？） |
| 字体 | Google Fonts（Space Grotesk + Inter） |
| 响应式 | 移动端适配（但以桌面为主） |

选择纯静态的理由：
- 单页，不需要框架
- Cloudflare Pages 直接部署，无构建步骤
- 加载极快，零依赖
- 后期可随时迁移到 Next.js/Vite
